Telling Claude "make this button bigger" doesn't say *which* button. You end up describing rectangles like a court witness.

[Avis](https://github.com/sryo/avis) lets you point at elements on a webpage. Claude Code reads the annotation in context: the actual node, its surroundings, the page state. Built on [perch](https://github.com/sryo/perch), a small AppleScript/JXA driver for Chrome and Safari that does the pointing-at-pixels part.
