Design files drift from the code they were supposed to represent. Components ship, the Figma file stays at v3, and the next designer inherits a fiction.

[Figma-CDP](https://github.com/sryo/figma-cdp) is a Claude Code skill that drives Figma over the Chrome DevTools Protocol. Claude reads your code, opens the file, and updates frames to match. The canvas catches up to the build instead of describing what it used to be.
